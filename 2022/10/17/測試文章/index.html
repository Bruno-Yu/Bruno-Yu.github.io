Hexo 部署GitHub注意地雷

發表於 2022-10-17 更新於 2022-10-29 分類於 hexo Disqus：
hexo-deployer-git 其預設指令就有 git push--f 與 git deploy 因此切記不要再自己重新再下git push 指令了，而自己的local repo 的git flow 與遠端的repo 內容由於 git push--f的特性，所以可以沒有直接關聯

特別注意

  • 本文參考自 Ray的文章: 試著學 Hexo - 部署你的第一個部落格
  • hexo-deployer-git 運作方式
  • 安裝完畢後要在 hexo /_config.yml 做修正 ( 注意: 不是在主題的 themes/_config.yml 做修正 )
  • 上傳指令

注意: 根據hexo 的官網表述 ( #3002 )

  • 此指令是包含 commit, push—force 等，所以hexo 的上傳不是自行下git push 完成，這點請注意
